# Tugas 4 - Movie CRUD

Project ini merupakan aplikasi CRUD data movie menggunakan Java.

## Fitur
- Create data movie
- Read data movie
- Update data movie
- Delete data movie
- Clear form
- Hitung nilai rating otomatis

## Teknologi
- Java
- Swing GUI
- MySQL
- JDBC
- Maven
- MVC
- OOP

## Konsep OOP
- Interface: CrudInterface
- Inheritance: Movie extends BaseEntity
- Encapsulation: penggunaan private attribute, getter, dan setter

## Database
Nama database: movie_db

Import database melalui file:

database/movie_db.sql

## Cara Menjalankan
1. Import database movie_db.sql ke MySQL
2. Buka project di NetBeans
3. Sesuaikan username dan password database pada file DatabaseConnection.java
4. Jalankan Main.java
